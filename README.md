# Christofell’s Fine Dining
A luxurious React Native experience that brings the elegance of gourmet cuisine to your fingertips.

## Overview
Christofell’s Fine Dining is a modern mobile application designed to simulate the ambiance and sophistication of a high-end dining experience.
From curated menus to category-based filtering, the app allows users to explore gourmet dishes, view stunning visuals, and even add their own culinary creations — all within an elegant dark-gold themed interface.
Built using React Native and TypeScript, this app blends refined aesthetics with seamless functionality, offering a smooth, intuitive, and visually rich user experience.

## Features
1. Welcome Screen: A refined entry point introducing Christofell’s signature aesthetic.
2. Home Screen: Displays all fine dining items with rich visuals, pricing, and descriptions.
3. Add Item Screen: Enables users to contribute new gourmet dishes dynamically.
4. Filter Screen: Filters menu items by course — Starters, Mains, or Desserts.
5. Elegant Theme: Jet black, antique gold, and ivory tones evoke a luxury dining feel.
6. Smooth Navigation: Powered by React Navigation for seamless screen transitions.
7. Typed Structure: Strong TypeScript integration for predictable and maintainable logic.

## Tech Stack
1. Language: TypeScript
2. Framework: React Native (Expo compatible)
3. Navigation: @react-navigation/native + @react-navigation/native-stack
4. UI Components: react-native, @react-native-picker/picker
5. Styling: Custom StyleSheet with dark-mode design constants

## How It Works
1. Home Screen: Showcases menu items with descriptions, price, and images.
2. Add Item Screen: Lets users add new dishes to the menu with name, price, category, and image.
3. Filter Screen: Allows selection between Starter, Main, or Dessert for precise browsing.
4. Typed Navigation — Ensures consistent, error-free transitions across the stack.
